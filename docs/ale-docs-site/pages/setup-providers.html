Setup Environment Provider

A provider boots the sandbox each task runs in. Set one up here, then configure a run.

Standing up the environment that serves the sandbox is the heaviest, most error-prone part of any agent benchmark — and ALE asks more of it than most. Tasks ship real, large professional software and their input data, and a single run can stretch for hours, so the platform also has to scale out across many sandboxes at once. ALE intends to make the process as easy and straightforward as possible. Google Cloud is the supported path today with 300 USD credit that's friendly enough to get you started; more cloud backends, plus a local path for debugging and development on your own hardware, are on the roadmap.

Whichever provider you pick, clone and install first:

git clone git@github.com:rdi-berkeley/agents-last-exam.git
cd agents-last-exam
uv sync --all-packages
